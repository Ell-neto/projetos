Manipulação de Arrays de Objetos - Questão 4

Objetivo: Crie um array de objetos pessoas, onde cada objeto representa uma pessoa com nome, idade, e cidade. Use for of para exibir as informações de cada pessoa no console.

function executarCodigo() {
    const pessoas = [
        { nome: 'Lan', idade: 33, cidade: 'Goiana' },
        { nome: 'Layne', idade: 30, cidade: 'João Pessoa' },
        { nome: 'Andreza', idade: 30, cidade: 'Recife' },
        { nome: 'Josenildo', idade: 24, cidade: 'Condado' },
        { nome: 'Ana', idade: 28, cidade: 'Recife' }
    ];

    for (const pessoa of pessoas) {
        console.log(\`Nome: \${pessoa.nome}, Idade: \${pessoa.idade}, Cidade: \${pessoa.cidade}\`);
    }
}
