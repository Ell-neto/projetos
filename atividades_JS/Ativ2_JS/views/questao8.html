Manipulação de Arrays de Objetos - Questão 8

Objetivo: Crie um array de objetos filmes, onde cada filme tem título, diretor e ano de lançamento. Use forEach para criar um novo array contendo apenas os títulos dos filmes.

function executarCodigo() {
    const filmes = [
        // O array será preenchido com os valores fornecidos pelo usuário
    ];

    const titulos = [];
    filmes.forEach(filme => {
        titulos.push(filme.titulo);
    });

    console.log(titulos);
}
